# EDU-SaaS (Flutter)

Multi-tenant education SaaS (mobile + web). This repo includes core scaffolding, authentication foundation, and owner dashboard skeleton.

## Getting Started

1. Install Flutter (3.22+) and Dart SDK (3.3+)
2. Enable web: `flutter config --enable-web`
3. Get deps: `flutter pub get`
4. Generate code: `dart run build_runner build -d`
5. Run web: `flutter run -d chrome`

> Note: This workspace includes Retrofit/Hive/JsonSerializable codegen. Backend endpoints are placeholders.
